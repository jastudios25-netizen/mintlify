---
title: "Metrics History"
api: "GET https://api.phanomcloud.online/bots/{id}/metrics/history"
description: "Returns up to 30 minutes of metrics history. One data point every 30 seconds (max 60 points)."
authMethod: "bearer"
---

## Path Parameters

<ParamField path="id" type="string" required>
  UUID of the bot.
</ParamField>

## Query Parameters

<ParamField query="minutes" type="number" default="30">
  How many minutes of history to return. Min: `1`, Max: `30`.
</ParamField>

<RequestExample>
```bash cURL
curl "https://api.phanomcloud.online/bots/3fa85f64-5717-4562-b3fc-2c963f66afa6/metrics/history?minutes=10" \
  -H "Authorization: Bearer sess_..."
```
</RequestExample>
