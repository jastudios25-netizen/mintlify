---
title: "Stream Metrics (SSE)"
api: "GET https://api.phanomcloud.online/bots/{id}/metrics/stream"
description: "Streams live resource usage metrics via Server-Sent Events (SSE). Emits a new data point every 30 seconds."
authMethod: "bearer"
---

## Path Parameters

<ParamField path="id" type="string" required>
  UUID of the bot.
</ParamField>

## Query Parameters

<ParamField query="token" type="string">
  Session token for browser clients (use instead of `Authorization` header).
</ParamField>

<CodeGroup>
```js Browser
const evtSource = new EventSource(
  "https://api.phanomcloud.online/bots/BOT_ID/metrics/stream?token=sess_..."
);

evtSource.onmessage = (e) => {
  const metrics = JSON.parse(e.data);
  console.log("CPU:", metrics.cpu_percent + "%");
  console.log("RAM:", metrics.ram_percent + "%");
};
```

```js Node.js
const res = await fetch(
  "https://api.phanomcloud.online/bots/BOT_ID/metrics/stream",
  { headers: { Authorization: "Bearer sess_..." } }
);

for await (const chunk of res.body) {
  const text = new TextDecoder().decode(chunk);
  // Parse SSE data lines
  for (const line of text.split("\n")) {
    if (line.startsWith("data: ")) {
      const metrics = JSON.parse(line.slice(6));
      console.log(metrics);
    }
  }
}
```
</CodeGroup>
