---
title: "Stream Logs (SSE)"
api: "GET https://api.phanomcloud.online/bots/{id}/logs/stream"
description: "Streams bot logs in real-time using Server-Sent Events (SSE). Sends a keep-alive ping every 15 seconds."
authMethod: "bearer"
---

## Path Parameters

<ParamField path="id" type="string" required>
  UUID of the bot.
</ParamField>

## Query Parameters

<ParamField query="token" type="string">
  Session token for browser clients. Use this instead of the `Authorization` header, since the native `EventSource` API does not support custom headers.
</ParamField>

## Event Format

Each event has a `log` string and a `type` field:

| Type | Description |
|------|-------------|
| `history` | Previous log lines sent on initial connection |
| `live` | New log lines streamed in real-time |

Keep-alive pings are sent as `event: ping` every 15 seconds.

<CodeGroup>
```js Browser (EventSource)
const evtSource = new EventSource(
  "https://api.phanomcloud.online/bots/BOT_ID/logs/stream?token=sess_..."
);

evtSource.onmessage = (e) => {
  const { log, type } = JSON.parse(e.data);
  console.log(`[${type}]`, log);
};

evtSource.onerror = () => {
  evtSource.close();
};
```

```js Node.js (fetch)
const res = await fetch(
  "https://api.phanomcloud.online/bots/BOT_ID/logs/stream",
  { headers: { Authorization: "Bearer sess_..." } }
);

for await (const chunk of res.body) {
  const text = new TextDecoder().decode(chunk);
  console.log(text);
}
```
</CodeGroup>

## Stream Output Example

```
data: {"log": "[10:00:01] Bot started", "type": "history"}

data: {"log": "[10:00:02] Connected to Discord", "type": "history"}

data: {"log": "[10:00:15] Command received: !ping", "type": "live"}

event: ping
data: {}
```
